# Computer Force — E-Commerce Website

A fully responsive multi-page e-commerce website built with HTML, CSS, and vanilla JavaScript for a fictional computer hardware and software retailer based in Parramatta, Sydney.

---

## About the Project

Computer Force is a front-end web project that simulates a real-world online tech store. It features a clean, professional design with a complete shopping cart experience — no frameworks, no dependencies, just pure HTML, CSS, and JavaScript.

The site was built as part of the ICTWEB520 / ICTWEB519 coursework and demonstrates practical skills in responsive web design, DOM manipulation, session storage, and multi-page navigation.

---

## Features

- **Responsive design** — fully optimised for mobile, tablet, and desktop
- **Mobile navigation** — hamburger slide-in menu and shopping bag panel
- **Live shopping cart** — add items, view quantities, remove products, and place orders
- **Session persistence** — cart data is preserved across page navigation using `sessionStorage`
- **Cart badge** — real-time item count displayed on the bag icon
- **Checkout panel** — slide-in cart panel with item list, total, and order placement on every page
- **72 products** across 12 categories, each with images, names, and prices
- **Registration form** with client-side validation
- **About page** with company info and contact details

## Tech Stack

- **HTML5** — semantic markup, accessibility attributes
- **CSS3** — custom properties, flexbox, media queries, CSS animations

## How to Run

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. No server required — runs entirely in the browser
